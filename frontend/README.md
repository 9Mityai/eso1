# EuroSecondOpinion Frontend MVP

Next.js (App Router) + TypeScript frontend for a premium telemedicine dashboard using a high-tech Swiss medical visual language.

## Stack

- Next.js App Router
- Tailwind CSS
- Lucide React
- Framer Motion

## Run

```bash
npm install
npm run dev
```

## Validate

```bash
npm run typecheck
npm run lint
npm run build
```
